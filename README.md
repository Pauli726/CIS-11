# CIS-11
Triple A project
CIS-11-Project
Test Score Calculator (LC‑3)

This project is a simple LC‑3 assembly program that takes five test scores from the user (00–99), stores them, and then shows the max, min, and average along with their letter grades. It also prints the letter grade for each score as you enter them. Basically, it’s just doing the usual grade‑calculation stuff but in LC‑3.

What the program does:
-Lets you type 5 scores
-Converts the ASCII digits into actual numbers
-Prints each score with its letter grade

Calculates:
-MAX
-MIN
-AVG

Prints the letter grade for those too
Uses a stack to push/pop the letter grades
Has a restart option so you can run it again without reloading
